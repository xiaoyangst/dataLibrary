# 目标检测的基本步骤

## 单目标检测和多目标检测

| 对比项     | 单目标检测           | 多目标检测                 |
| ---------- | -------------------- | -------------------------- |
| 目标数量   | 1                    | 多                         |
| 类别数量   | 1 或少数             | 多类（通常 >2）            |
| 输出形式   | 单一框 / 单一类      | 多个框 + 类别 + 置信度     |
| 模型复杂度 | 低                   | 高                         |
| 应用场景   | 人脸检测、车牌识别等 | 自动驾驶、视频监控、安防等 |
